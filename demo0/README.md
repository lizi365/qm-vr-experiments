# Quantum VR — Demo 0（链路验证：WebXR旋转立方体）

## 这个demo的目的

不测性能、不测视觉效果，只确认一件事：**WebXR + Three.js + Cloudflare Pages 这条技术链路能不能跑通**——从 Pico 企业版头显浏览器打开链接，能不能正常进入VR模式，能不能看到一个稳定渲染的3D物体。

如果这个demo在头显上能正常跑起来，说明：
- Pico企业版浏览器对 WebXR API 的支持没问题
- Cloudflare Pages 部署的纯静态HTML能被头显正常访问和加载
- Three.js 从 unpkg CDN 加载 importmap 依赖没有网络/兼容性问题

## 场景里有什么

- **正前方一个持续旋转的立方体**：唯一的测试对象，简单到不会引入任何性能干扰
- **头顶悬浮面板**：实时显示FPS，头显里直接能看到，不用摘头显看电脑
- **桌面模式**：不戴头显也能在浏览器里直接看效果（右上角DOM文字显示FPS）

## 操作方式

没有交互操作——打开链接、点击"Enter VR"按钮即可，之后什么都不用做，看立方体转、看FPS是否稳定就行。
